# The Prosphect
somewhere in the very near, yet distant future... 

you, yes, YOU. are in a dystopian state-controlled suburban sector portrayed as marrow vantage. a humble nonchalant fellow with an unhealthy obsession for anything NOIR. all human interaction, media, and digital networking are coerced through a single unoptimized though laced with the sense of modernity, intranet portal: **THE PROSPHECT**. 
knowing your governmental predicament, you still somehow decide it is a wonderous idea to run an illegal, un-indexed alpha executable downloaded from the back alleys of the prosphect directory. ultimatley triggering a catastrophic localized system crash! but that is only where the real problem begins.
